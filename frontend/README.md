# frontend

## Purpose
This folder (`frontend`) contains files for frontend within the Home-Services-Marketplace project.

## Files
- `auth.html`: HTML page/template for auth.
- `booking.html`: HTML page/template for booking.
- `bookings.html`: HTML page/template for bookings.
- `cart.html`: HTML page/template for cart.
- `compare.html`: HTML page/template for compare.
- `index.html`: HTML page/template for index.
- `package.json`: JSON configuration/data for package.
- `profile.html`: HTML page/template for profile.
- `provider-dashboard.html`: HTML page/template for provider dashboard.
- `service.html`: HTML page/template for service.

## Note
JSON and binary files are documented in this README because inline comments are unsafe for those formats.
